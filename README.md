# To Do List

> Welcome to my to-do project

![screenshot](./screenshot.png)

- Major languages : Html, CSS, JS

### Setup
To install all required modules:
- `npm install`

You can run the project on  `http://localhost:8080/` with webpack by writing the below code:
- `npm start`
